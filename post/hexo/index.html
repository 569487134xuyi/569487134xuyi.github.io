github+hexo 搭建

github+hexo 搭建

github 部分

github.com
github.com 注册
你的 github 地址：github.com/username（你的注册名）
start a project 按钮新建一个 repositories（仓库）
仓库名：username (和上面一样，这里不要用别的).github.io
creat repository 创建仓库

git 及 node.js 安装

git
一直下一步不多说 安装完了右键 git bash

git config user.name 'github用户名'
git config user.email '邮箱'

node.js
下载完一直下一步安装

node -v
npm -v
git -version

看一下 node.js,npm,git 版本信息
以上没成功，环境变量自己再看下

hexo 部分

hexo.io
右键 git bash here
输入以下

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

安装出现 warn 都不要管
上面会自动创建一个 blog 文件夹，进入 blog 并启动 hexo
地址在本地 4000 端口
http://localhost:4000/

在 blog/source/_posts 下有自动创建的一篇 helloworld 的初始化文章

如果想要添加新的文章，可以在命令行输入 hexo new 'filename',就会生成一个新的.md 文件，对它编辑就可以了

上传至 github

开配置文档_config.yml，对它做如下修改

deploy:
  type: git
  repo: git@github.com:username/username.github.io.git
  branch: master

git bash 中输入

npm install hexo-deployer-git --save

安装 git 插件 再输入

hexo g
hexo d

开始上传
打开 username.github.io blog 就部署好了

结束
